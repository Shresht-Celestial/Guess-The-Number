# Guess-The-Number
🎯 A fun interactive number guessing game built using HTML and JavaScript with real-time feedback and score tracking.

🎯 Guess The Number Game

A simple and interactive number guessing game where the user tries to guess a randomly generated number within a limited number of attempts.

🚀 Features
🎮 Game Logic
Random number generation
User input handling

Feedback:
Too high 📈
Too low 📉
Correct ✅

❤️ Score System
Starts with a maximum score (e.g., 20)
Decreases on wrong guesses
High score tracking

🔄 Reset Functionality
Restart the game anytime

⚠️ Input Validation
Handles:
Empty input
Invalid numbers

🛠️ Tech Stack
HTML5
Vanilla JavaScript
